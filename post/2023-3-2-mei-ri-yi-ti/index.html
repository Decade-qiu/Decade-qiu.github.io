面试题 05.02. 二进制数转字符串 - 力扣（Leetcode）

二进制数转字符串。给定一个介于0和1之间的实数（如0.72），类型为double，打印它的二进制表达式。如果该数字无法精确地用32位以内的二进制表示，则打印“ERROR”。

示例1:

 输入：0.625
 输出："0.101"

示例2:

 输入：0.1
 输出："ERROR"
 提示：0.1无法被二进制准确表示

思路

考察计算机硬件基础的浮点数二进制存储。

对于小数点后的第 iii 位， 如果为1，则它可以算作 12i\frac{1}{2^i}2i1​ 。否则就是0。

该题只需要遍历30次，依次加上每一位的价值，最后看num是不是为0.

代码

class Solution:
    def printBin(self, num: float) -> str:
        ans = "0."
        for i in range(1, 31):
            cur = 1.0 / (1 << i)
            if num >= cur:
                num -= cur
                ans += "1"
            else: ans += "0"
            if num == 0: break
        return ans if num==0 else "ERROR"

复杂度分析

  • 时间复杂度: O(30)O(30)O(30) 。
  • 空间复杂度: O(1)O(1)O(1) 。
