Leetcode 1139. 最大的以 1 为边界的正方形

Leetcode

1139. 最大的以 1 为边界的正方形 - 力扣（Leetcode）

给你一个由若干 0 和 1 组成的二维网格 grid，请你找出边界全部由 1 组成的最大 正方形 子网格，并返回该子网格中的元素数量。如果不存在，则返回 0。

示例 1：

输入：grid = [[1,1,1],[1,0,1],[1,1,1]]
输出：9

示例 2：

输入：grid = [[1,1,0,0]]
输出：1

提示：

  • 1 <= grid.length <= 100
  • 1 <= grid[0].length <= 100
  • grid[i][j] 为 0 或 1

思路

  1. 注意到题目n范围。可以在 O(n3)O(n^3)O(n3) 复杂度时间过。于是暴力遍历每个点，枚举正方形长度，依次判断是否可行。但此时时间复杂度已经 O(n3)O(n^3)O(n3)，我们需要 O(1)O(1)O(1) 时间内判断。于是可以引入前缀和，计算正方形四条边上数值的和是否等于正方形的周长。如果是，说明四周都为1，符合题意。

代码

class Solution:
    def largest1BorderedSquare(self, g: List[List[int]]) -> int:
        n, m, ans = len(g), len(g[0]), 0
        col, row = [[0]*(m+1) for i in range(n+1)], [[0]*(m+1) for i in range(n+1)]
        for i in range(1, n+1):
            for j in range(1, m+1): row[i][j] = row[i][j-1]+g[i-1][j-1]
        for i in range(1, m+1):
            for j in range(1, n+1): col[j][i] = col[j-1][i]+g[j-1][i-1]
        for i in range(1, n+1):
            for j in range(1, m+1):
                for k in range(1, min(n-i+1, m-j+1)+1):
                    if row[i][j+k-1]-row[i][j-1]+row[i+k-1][j+k-1]-row[i+k-1][j-1]+col[i+k-1][j]-col[i-1][j]+col[i+k-1][j+k-1]-col[i-1][j+k-1] == 4*k:
                        ans = max(ans, k*k)
        return ans

复杂度分析

  • 时间复杂度: O(n3)O(n^3)O(n3) 。
  • 空间复杂度: O(n3)O(n^3)O(n3) 。
