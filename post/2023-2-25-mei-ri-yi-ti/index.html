Leetcode 1247. 交换字符使得字符串相同

CodeForces B. Recover the String

CodeForces

Problem - 708B - Codeforces

对于 01 字符串 s，定义 f(x,y) 表示子序列 [x,y] 在 s 中的出现次数。
输入 f(0,0), f(0,1), f(1,0) 和 f(1,1)，范围在 [0,1e9]。
请构造任意一个满足输入的非空字符串 s。
如果不存在，输出 Impossible。

思路

  1. 设 有 x 个 0 , 那 么 00 的 组 合 就 是 x ∗ ( x − 1 ) / 2。根据 f(0,0) 可以求出 0 的个数 c0，因为 f(0,0) = C(c0,2)；同理可求出 1 的个数 c1。

  2. f(0,1) + f(1,0) = c0 * c1

代码

def solve():
    a, b, c, d = list(map(int, input().split()))
    n, pre = 1000001, 0
    dic = {0:1}
    for i in range(2, n):
        pre += i-1
        dic[pre] = i
    c00, c11 = dic.get(a, -1), dic.get(d, -1)
    if a+b+c == 0 and c11 != -1:
        print(c11*'1')
        return
    if b+c+d == 0 and c00 != -1:
        print(c00*'0')
        return
    if c00==-1 or c11==-1 or b+c != c00*c11:
        print("Impossible")
    else:
        ans = ""
        for i in range(b//c11): ans += '0'
        for i in range(c11-b%c11): ans += '1'
        if c00-b//c11 > 0: 
            ans += '0'
            for i in range(b%c11): ans += '1'
            for i in range(c00-b//c11-1): ans += '0'
        print(ans)

复杂度分析

  • 时间复杂度: O(1e6)O(1e6)O(1e6) 。
  • 空间复杂度: O(1e6)O(1e6)O(1e6) 。

Leetcode

1247. 交换字符使得字符串相同 - 力扣（Leetcode）

有两个长度相同的字符串 s1 和 s2，且它们其中 只含有 字符 "x" 和 "y"，你需要通过「交换字符」的方式使这两个字符串相同。

每次「交换字符」的时候，你都可以在两个字符串中各选一个字符进行交换。

交换只能发生在两个不同的字符串之间，绝对不能发生在同一个字符串内部。也就是说，我们可以交换 s1[i] 和 s2[j]，但不能交换 s1[i] 和 s1[j]。

最后，请你返回使 s1 和 s2 相同的最小交换次数，如果没有方法能够使得这两个字符串相同，则返回 -1 。

示例 1：

输入：s1 = "xx", s2 = "yy"
输出：1
解释：
交换 s1[0] 和 s2[1]，得到 s1 = "yx"，s2 = "yx"。

示例 2：

输入：s1 = "xy", s2 = "yx"
输出：2
解释：
交换 s1[0] 和 s2[0]，得到 s1 = "yy"，s2 = "xx" 。
交换 s1[0] 和 s2[1]，得到 s1 = "xy"，s2 = "xy" 。
注意，你不能交换 s1[0] 和 s1[1] 使得 s1 变成 "yx"，因为我们只能交换属于两个不同字符串的字符。

提示：

  • 1 <= s1.length, s2.length <= 1000
  • s1, s2 只包含 'x' 或 'y'。

思路

每个字符串中因为只有两种字符x,y。所以s1,s2相同位置字符对只有4种

  • xx
  • yy
  • xy
  • yx

要求操作次数最小，所以前两种已经匹配的就不需要再交换了。

对于剩下两个，如果存在两个xy，或者两个yx，例1已经给出交换方案只需要交换1次就行。

但存在一个xy和一个yx，例2也指出只需交换2次。

于是，优先将xy和yx分别内部消化，多出来的再按例2那样消化。

代码

class Solution {
    public int minimumSwap(String s1, String s2) {
        int n = s1.length(), m = s2.length(), xy = 0, yx = 0;
        if (n != m) return -1;
        for (int i = 0;i < n;i++){
            char a = s1.charAt(i), b = s2.charAt(i);
            if (a == b) continue;
            else if (a == 'x') xy++;
            else yx++;
        }
        if ((xy+yx)%2 == 1) return -1;
        return xy/2+yx/2+(xy%2+yx%2);
    }
}

复杂度分析

  • 时间复杂度: O(n)O(n)O(n) 。
  • 空间复杂度: O(1)O(1)O(1) 。
