162. 寻找峰值

Leetcode 寻找峰值

峰值元素是指其值严格大于左右相邻值的元素。

给你一个整数数组 nums，找到峰值元素并返回其索引。数组可能包含多个峰值，在这种情况下，返回 任何一个峰值 所在位置即可。

你可以假设 nums[-1] = nums[n] = -∞ 。

你必须实现时间复杂度为 O(log n) 的算法来解决此问题。

示例 1：

输入：nums = [1,2,3,1]
输出：2
解释：3 是峰值元素，你的函数应该返回其索引 2。

示例 2：

输入：nums = [1,2,1,3,5,6,4]
输出：1 或 5 
解释：你的函数可以返回索引 1，其峰值元素为 2；
     或者返回索引 5， 其峰值元素为 6。

提示：

  • 1 <= nums.length <= 1000
  • -231 <= nums[i] <= 231 - 1
  • 对于所有有效的 i 都有 nums[i] != nums[i + 1]

思路

  1. 常规做法很简单，遍历一遍，找到符合条件的下标即可
  2. 题目要求O(log n)，直接想到分治。
  3. 但题解里都是用二分做的，没看到用分治的。因为二分更简单⭕

代码

/**
 * @param {number[]} nums
 * @return {number}
 */
var findPeakElement = function(nums) {
    let n = nums.length;
    let find = (l, r) => {
        if (l == r){
            if ((l==0||nums[l]>nums[l-1]) && (l==n-1||nums[l]>nums[l+1])) return l;
            return -1;
        }
        let mid = l+r >> 1;
        let x = find(l, mid), y = find(mid+1, r);
        if (x != -1) return x;
        else if (y != -1) return y;
        else return -1;
    };
    return find(0, n-1);
};

复杂度分析

  • 时间：O(log⁡n\log nlogn)。
  • 空间：O($$1$$)。
