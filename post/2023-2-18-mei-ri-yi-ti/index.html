Leetcode 1237. 找出给定方程的正整数解

Leetcode

1237. 找出给定方程的正整数解

给你一个由若干 0 和 1 组成的二维网格 grid，请你找出边界全部由 1 组成的最大 正方形 子网格，并返回该子网格中的元素数量。如果不存在，则返回 0。

示例 1：

给你一个函数  f(x, y) 和一个目标结果 z，函数公式未知，请你计算方程 f(x,y) == z 所有可能的正整数 数对 x 和 y。满足条件的结果数对可以按任意顺序返回。

尽管函数的具体式子未知，但它是单调递增函数，也就是说：

f(x, y) < f(x + 1, y)
f(x, y) < f(x, y + 1)输入：grid = [[1,1,1],[1,0,1],[1,1,1]]
输出：9

示例 2：

输入：grid = [[1,1,0,0]]
输出：1

提示：

  • 1 <= grid.length <= 100
  • 1 <= grid[0].length <= 100
  • grid[i][j] 为 0 或 1

思路

  1. 暴力。
  2. 双指针。题目中已说单调性 => 如果 f(x,y)<zf(x, y) < zf(x,y)<z，则对任意 y1<y,f(x,y1)<zy1 < y, f(x, y1) < zy1<y,f(x,y1)<z 。此时让 yyy 减少不可行，只能让 xxx 增加。并且增加 xxx 后，假设x1<x2，且f(x1,y1)=f(x2,y2)=z，显然有y1>y2假设 x_1 < x_2，且 f(x_1, y_1) = f(x_2, y_2) = z，显然有 y_1 > y_2假设x1​<x2​，且f(x1​,y1​)=f(x2​,y2​)=z，显然有y1​>y2​ 。所以 yyy 只能依次减小，不需要从头枚举。

代码

class Solution {
    public List<List<Integer>> findSolution(CustomFunction c, int z) {
        List<List<Integer>> ans = new ArrayList<>();
        int i = 1, j = 1000;
        while (i<=1000 && j >= 1){
            int r = c.f(i, j);
            if (r == z){
                List<Integer> tp = new ArrayList<>();
                tp.add(i); tp.add(j);
                ans.add(tp);
                ++i;
            }else if (r > z) --j;
            else ++i;
        }
        return ans;
    }
}

复杂度分析

  • 时间复杂度: O(n+m)O(n+m)O(n+m) 。
  • 空间复杂度: O(1)O(1)O(1) 。
