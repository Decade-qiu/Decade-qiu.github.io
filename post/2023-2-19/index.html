Leetcode 1792. 最大平均通过率

Leetcode

1792. 最大平均通过率 - 力扣（Leetcode）

一所学校里有一些班级，每个班级里有一些学生，现在每个班都会进行一场期末考试。给你一个二维数组 classes ，其中 classes[i] = [passi, totali] ，表示你提前知道了第 i 个班级总共有 totali 个学生，其中只有 passi 个学生可以通过考试。

给你一个整数 extraStudents ，表示额外有 extraStudents 个聪明的学生，他们 一定 能通过任何班级的期末考。你需要给这 extraStudents 个学生每人都安排一个班级，使得 所有 班级的 平均 通过率 最大 。

一个班级的 通过率 等于这个班级通过考试的学生人数除以这个班级的总人数。平均通过率 是所有班级的通过率之和除以班级数目。

请你返回在安排这 extraStudents 个学生去对应班级后的 最大 平均通过率。与标准答案误差范围在 10-5 以内的结果都会视为正确结果。

示例 1：

输入：classes = [[1,2],[3,5],[2,2]], extraStudents = 2
输出：0.78333
解释：你可以将额外的两个学生都安排到第一个班级，平均通过率为 (3/4 + 3/5 + 2/2) / 3 = 0.78333 。

示例 2：

输入：classes = [[2,4],[3,9],[4,5],[2,10]], extraStudents = 4
输出：0.53485

提示：

  • 1 <= classes.length <= 10^5
  • classes[i].length == 2
  • 1 <= passi <= totali <= 10^5
  • 1 <= extraStudents <= 10^5

思路

设某个班级的人数为 totalt o t a ltotal ，其中通过考试的人数为 pass，那么给这个班级安排一个额外通过考试的学生，其通过率会增加：pass+1 total +1−pass total \frac{p a s s+1}{\text { total }+1}-\frac{p a s s}{\text { total }} total +1pass+1​− total pass​。

所以每次贪心选择具有最大增量的那个班级就行了。这里用优先队列来实现，循环extraStudents次。

代码

class Solution {
    public double maxAverageRatio(int[][] c, int e) {
        int n = c.length;
        PriorityQueue<double[]> pq = new PriorityQueue<>(
            (x, y)->{
                double a = (x[0]+1)/(x[1]+1)-x[0]/x[1];
                double b = (y[0]+1)/(y[1]+1)-y[0]/y[1];
                return Double.compare(b, a);
            }
        );
        for (int[] cur : c) pq.add(new double[]{cur[0], cur[1]});
        for (int i = 0;i < e;i++){
            double[] cur = pq.poll();
            cur[0]++; cur[1]++;
            pq.add(cur);
        }
        double ans = 0;
        while (!pq.isEmpty()) {
            double[] cur = pq.poll();
            ans += cur[0]/cur[1];
        }return ans/n;
    }
}

复杂度分析

  • 时间复杂度: O(mlog⁡n)O(m\log n)O(mlogn) 。
  • 空间复杂度: O(n)O(n)O(n) 。
