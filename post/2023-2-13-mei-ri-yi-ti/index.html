Leetcode 1138. 字母板上的路径

Leetcode

1234. 替换子串得到平衡字符串 - 力扣（Leetcode）

有一个只含有 'Q', 'W', 'E', 'R' 四种字符，且长度为 n 的字符串。

假如在该字符串中，这四个字符都恰好出现 n/4 次，那么它就是一个「平衡字符串」。

给你一个这样的字符串 s，请通过「替换一个子串」的方式，使原字符串 s 变成一个「平衡字符串」。

你可以用和「待替换子串」长度相同的 任何 其他字符串来完成替换。

请返回待替换子串的最小可能长度。

如果原字符串自身就是一个平衡字符串，则返回 0。

示例 1：

输入：s = "QWER"
输出：0
解释：s 已经是平衡的了。

示例 2：

输入：s = "QQWE"
输出：1
解释：我们需要把一个 'Q' 替换成 'R'，这样得到的 "RQWE" (或 "QRWE") 是平衡的。

示例 3：

输入：s = "QQQW"
输出：2
解释：我们可以把前面的 "QQ" 替换成 "ER"。

示例 4：

输入：s = "QQQQ"
输出：3
解释：我们可以替换后 3 个 'Q'，使 s = "QWER"。

提示：

  • 1 <= s.length <= 10^5
  • s.length 是 4 的倍数
  • s 中只含有 'Q', 'W', 'E', 'R' 四种字符

思路

  1. 暴力想法：遍历每个子字符串长度，再利用滑动窗口看是否满足题意。

  2. 优化：子字符串越长，越可能满足题意。满足二分地单调性。

    即想到二分子字符串长度，再利用滑动窗口判断是否可行。

    在滑窗判断时，首先需要知道一开始所有字符出现次数。当删去滑窗内的字符后，所有字符出现次数都小于 n/4n/4n/4，说明可行；反之，说明不行。因为如果滑窗外有字符出现次数大于 n/4n/4n/4，无论滑窗内字符怎么更改，都不能让这个字符总次数等于 n/4n/4n/4。

代码

class Solution {
    Map<Character, Integer> mp = new HashMap<>();
    int n, k, cnt[] = new int[4];
    public boolean check(String s, int m) {
        int[] tp = Arrays.copyOf(cnt, cnt.length);
        for (int i = 0, j = 0;j < n;i++) {
            while (j < n && j - i != m) tp[mp.get(s.charAt(j++))]--;
            if (tp[0] <= k && tp[1] <= k && tp[2] <= k && tp[3] <= k)
                return true;
            tp[mp.get(s.charAt(i))]++;
        }
        return false;
    }
    public int balancedString(String s) {
        mp.put('Q', 0);mp.put('W', 1);
        mp.put('E', 2);mp.put('R', 3);
        n = s.length(); k = n / 4;
        for (int i = 0; i < n; i++) cnt[mp.get(s.charAt(i))]++;
        if (cnt[0] == k && cnt[1] == k && cnt[2] == k) return 0;
        int l = 1, r = n;
        while (l <= r) {
            int m = l + r >> 1;
            if (check(s, m)) r = m - 1;
            else l = m + 1;
        }
        return l;
    }
}

复杂度分析

  • 时间复杂度：O(nlog⁡n)O(n \log n)O(nlogn)。 nnn 是给定字符串长度。
  • 空间复杂度： O(n)O(n)O(n) 。

Tips

最小可能长度这类字眼也是二分答案的关键词，只要判断出答案具有单调性就可以愉快AC了。
