类型（type） 说明
feat 添加新功能、新特性
fix 修复 Bug
perf 优化性能（不改变功能前提下提升性能）
refactor 重构代码（不改变功能与行为）
docs 文档修改
style 代码格式调整（不包括 CSS 样式修改，例如分号、缩进等）
test 新增或修改测试用例
build 影响项目构建或依赖项的修改
revert 回退到上一次提交
ci 持续集成相关配置或文件修改
chore 其他更新或维护（不属于以上类型）
release 发布新版本
workflow 工作流（如 GitHub Actions、CI/CD）相关文件修改
